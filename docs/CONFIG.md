# Configuration

Environment variables (prefix `ODM_`):

- `DATA_DIR` (default `/data`)
- `CONFIG_DIR` (default `/data/config`)
- `INGEST_DIR` (default `/data/ingest`)
- `LIBRARY_DIR` (default `/data/library`)
- `FAILED_DIR` (default `/data/failed`)
- `TMP_DIR` (default `/data/tmp`)
- `SCAN_ENABLED` (default `true`)
- `SCAN_INTERVAL_SECONDS` (default `15`)  (reserved for future polling mode)
- `TESSERACT_LANG` (default `eng`)
- `PDF_TEXT_MIN_CHARS` (default `25`)
